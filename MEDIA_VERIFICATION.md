# Media/Images — финальная проверка перед следующим этапом

Четыре проверки. Если все ✓ — этап окончательно завершён.

| # | Проверка | Ожидание |
|---|----------|----------|
| 1 | **Upload без токена** | `POST /upload/image` без заголовка `Authorization` → **401** |
| 2 | **Upload с токеном** | `POST /upload/image` с `Authorization: Bearer <token>` и полем `image` → **200**, в ответе `image_url` |
| 3 | **Отправка только картинки** | Выбрать фото, не вводить текст, отправить → сообщение с картинкой в чате |
| 4 | **Отправка картинки с текстом** | Выбрать фото, ввести текст, отправить → в пузыре картинка и текст |

**Перед проверками:** backend запущен (`go run .` в `backend/`), пользователь залогинен (для п. 2–4).

**Автопроверка:** из каталога `backend/` выполнить:
```bash
go run ./cmd/verify_media http://localhost:8080
```
(если backend на другом порту — указать его в URL). Все 4 пункта проверяются автоматически.
