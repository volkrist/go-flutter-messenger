Эти команды запускать ТОЛЬКО на сервере (SSH: alex@89.167.112.246), не в PowerShell на ПК.

cd /opt/go-flutter-messenger
grep FCM .env
docker compose build --no-cache backend
docker compose up -d backend
docker logs go-flutter-messenger-backend 2>&1 | head -20
